Usage Guidelines

Color Usage

  • Void Black: Primary background color for depth and contrast
  • Sunset Orange: Call-to-action buttons, links, and attention elements
  • Moody Purple: Secondary elements, gradients, and accents
  • Forest Green: Success states, nature-themed elements
  • Satin Sheen Gold: Premium elements, headings, and highlights
  • Lunar Ash: Primary text color for optimal readability

Typography Hierarchy

  • H1: Use Display font (Aquoos 3D) for hero titles and main branding
  • H2-H3: Use Heading font (Rubik Glitch) for section titles
  • Navigation & UI: Use Primary font (JabbingDemoTexture) for interface elements
  • Body Text: Use Body font (Space Mono) for readability and content
  • Accents: Use Accent font (Genniro) for quotes and special callouts
  • Code/Technical: Use Special font (Spectra Volt) for technical content
